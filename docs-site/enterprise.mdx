## Custom validation rules

Enterprise plans can extend the engine with domain rules unique to your
organization — proprietary bounds, internal conservation constraints, or
company-standard acceptance criteria — evaluated alongside the built-in 287
checks.

```python
from simapi.plugins import validator, Check

@validator(domain="aerodynamics")
def internal_drag_ceiling(df, conditions):
    limit = conditions.get("program_cd_ceiling", 0.45)
    over = (df["drag_coefficient"] > limit).sum()
    return Check(
        name="program_drag_ceiling",
        ok=over == 0,
        detail=f"{over} trials exceed program Cd ceiling {limit}",
        severity="failed",
    )
```
